# Spring Boot Template for WebSocket

- This template is intended for the WebSocket functionality only
- Kafka is included in this template
- Sample classes are included
- **Intended for personal use only as this does not include complete features like JHipster**

# Technologies Used:

- Kotlin
- Spring Boot 3
- Spring Actuator
- Spring Kafka
- Spring Retry
- Spring WebFlux
- Spring WebSocket
- SpringDoc OpenAPI
- AWS CDK
- Docker
- JUnit
- Kubernetes
- MapStruct
- Micrometer Tracing
- OpenAPI
- Prometheus
- Zipkin

# Spring WebSocket Template

### To run the code:

`./gradlew bootRun`

### To run tests:

`./gradlew test`

### To build a JAR file:

`./gradlew build -x test`
